Batch processing — list of places one downloaded folder per place

Enter a list of places: each row has a free-form name (that of the generated folder) and a location (address with autocomplete, coordinates " lat, lon ", or MGRS). The Verify button geocodes and confirms each location. Generation downloads, for each place, a folder named after it containing: the satellite and OpenTopoMap views at the checked zooms (one image per zoom), data.json and a report.pdf (text: coordinates & links, no image).

Annotate downloaded images — North & scale

Drop images (satellite captures, maps…). The scale is read from the file name (e.g. " …_200m.webp ", " …_1.5km.png "), or otherwise from the PNG metadata (for older PNG captures). The tool's exports are in WebP format. .crdownload files (interrupted but complete downloads) are accepted. A North arrow and a scale bar are added; the value stays editable image by image.
